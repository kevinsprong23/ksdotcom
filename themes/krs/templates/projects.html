Links to miscellaneous side projects:

Dwelp solver
Browser-based solver for the mobile game Dwelp. Game logic as a javascript module, driver scripts using d3.
BUDA graph
Graph analysis using BUDA rosters dating back to 1999. Featured Python's requests + multiprocessing, and an Angular.js-based search capability.
NFL by School
Makeup of NFL by school attended over time for the top 20 schools by NFL representation since 1990.
Filterable d3 Map
d3 + topoJson plot of passenger enplanement data at U.S. airports in 2013, with a range filter and dynamic size updating.
GDP By Industry
ggplot2 plot of changes in composition of U.S. GDP over time for selected industries. Data covers 1997 - 2013.
State Metrics
2013 Federal spending and tax revenues by state, plotted against population and Obama's 2012 victory margin. Interactivity and mouseovers via d3-tip and d3.js.
Housing Costs
Housing costs versus incomes for the greater Boston area. Inflation-adjusted median incomes, monthly mortgage costs, and monthly rents for five Boston-area counties.
Game bots
I built a Java app using Alpha-Beta pruning to beat the games 2048 and Threes!. They do pretty well, I have a writeup about it on this site and the code is also available on my github page. Follow this link to see a brief overview of the project.
This website
This was my first attempt at putting up a website from scratch - HTML/CSS, web hosting, domain name registration, site layout, etc. It started as a static page just to host my resume, then I slowly added responsive design in some places and finally integrated everything into the Pelican workflow.
